# Block Explorers

> Arc's block explorers to view transactions, verify contracts, and analyze onchain data using Etherscan-compatible APIs.

Block explorers let you view, verify, and analyze transactions, addresses,
contracts, and other onchain data on Arc.

## Providers

| Provider                                     | Description                                                                                                                                                                                        |
| :------------------------------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [**Blockscout**](https://www.blockscout.com) | Open-source explorer offering contract verification, token tracking, and API endpoints. A Blockscout explorer is available for [Arc Testnet](https://testnet.arcscan.app). |

> Arc's explorers are continuously expanding: developers can use standard
> Etherscan-compatible APIs to query data and verify contracts seamlessly.
